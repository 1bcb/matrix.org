The native Android version of the [Riot](/docs/projects/client/riot) web client. Riot is a glossy client with focus on performance and usability.

The code is available from [GitHub](https://github.com/vector-im/riot-android), and the app is available from the [Google Play store](https://play.google.com/store/apps/details?id=im.vector.app) and [F-Droid](https://f-droid.org/packages/im.vector.alpha/).

Riot Android is currently being rewritten with a new user experience. The beta version of the new app is called [RiotX](/docs/projects/client/riotx). Go check it out!
